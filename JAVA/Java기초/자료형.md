# 자료형

### 정수 자료형

- `byte`
  - 1바이트
  - 동영상, 음악 파일, 실행 파일을 처리할 때 사용
- `short`
  - 2바이트
  - C/C++과 호환 시 사용
    - Ex) 라이브러리 개발
- `int`
  - 4바이트
- `long`
  - 8바이트
  - 숫자의 뒤에 `L`또는 `l`을 붙여 `long` 자료형임을 표시



### 실수 자료형

- 부동소수점 방식으로 소수를 표현
  - `1.m * 2^n`



- `float`
  - 32비트
  - 부호비트: 1, 지수부: 8비트, 가수부: 23비트
  - 숫자의 뒤에 `F`또는 `f`을 붙여 `long` 자료형임을 표시
- `double`
  - 64비트
  - 부호비트: 1, 지수부: 11비트, 가수부: 52비트



- 부동소수점 방식의 오류
  - 지수부가 오류를 표현 할 수 없기 때문에 약간의 오차가 발생
