## 기본 명령어

- 명령어 --help
  - 해당 명령어의 사용법, 옵션 등을 알려줌

- `ls`
  - List
  - 파일 목록 보기
  - 윈도우의 `dir`
- `cd`
  - Chage Directory
  - 디렉토리 이동 명령어
- `pwd`
  - Print Working Directory
  - 현재 디렉토리의 전체 경로
- `touch`
  - 크기가 0인 새파일 생성
  - 파일이 존재한다면 vkdlfdml
